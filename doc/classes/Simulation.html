Simulation Class

Defined in: src/main/js/jquery-simula.js:855
Module: jQuery.simula

Creates a Simulation. Moves a pixel per millisecond. Note that the internal $element and elementPosition only reflect the current state during construction/chaining of the Simulation. This means that these stay the same during execution.
