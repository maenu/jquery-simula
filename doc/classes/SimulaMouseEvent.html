button

Number

Defined in src/main/js/jquery-simula.js:390

During mouse events caused by the depression or release of a mouse button, button is used to indicate which mouse button changed state. The values for button range from zero to indicate the left button of the mouse, one to indicate the middle button if present, and two to indicate the right button. For mice configured for left handed use in which the button actions are reversed the values are instead read from right to left. Use $.simula.MOUSE.BUTTON.[LEFT | MIDDLE | RIGHT].

Default: $.simula.MOUSE.BUTTON.LEFT
